API Documentation for Profile Management

Base URL

http://localhost:3001/api

1. Get All Profiles

Endpoint: GET /profiles

Description: Retrieve a list of all profiles stored on the server.

Request Example

GET http://localhost:3001/api/profiles

Response Example

{
  "data": [
    {
      "id": "123",
      "username": "JohnDoe",
      "bio": "Hello, this is my bio.",
      "socialMedia": {
        "facebook": "john_doe_fb",
        "twitter": "john_doe_twitter",
        "instagram": "john_doe_ig"
      }
    }
  ]
}

2. Save or Update Profile

Endpoint: POST /save-profile

Description: Save or update a profile based on the `username` parameter in the request body.

Request Example

{
  "username": "JohnDoe",
  "bio": "This is my bio",
  "socialMedia": {
    "facebook": "john_doe_fb",
    "twitter": "john_doe_twitter",
    "instagram": "john_doe_instagram",
    "snapchat": "john_doe_snapchat",
    "linkedin": "john_doe_linkedin"
  }
}

Response Example

{
  "success": true,
  "message": "Profile saved successfully"
}

3. Get Profile by Username

Endpoint: GET /profiles/{username}

Description: Retrieve the details of a profile based on the `username` in the URL.

Request Example

GET http://localhost:3001/api/profiles/johndoe

Response Example

{
  "data": {
    "id": "123",
    "username": "JohnDoe",
    "bio": "This is my bio",
    "socialMedia": {
      "facebook": "john_doe_fb",
      "twitter": "john_doe_twitter",
      "instagram": "john_doe_instagram",
      "snapchat": "john_doe_snapchat",
      "linkedin": "john_doe_linkedin"
    }
  }
}

4. Delete Profile by Username

Endpoint: DELETE /delete-profile/{username}

Description: Deletes a profile based on the `username` parameter in the URL.

Request Example

DELETE http://localhost:3001/api/delete-profile/johndoe

Response Example

{
  "success": true,
  "message": "Profile deleted successfully"
}
